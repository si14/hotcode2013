О понятиях

Logic variable

  • обозначает «какое-то» значение
  • возможно, значение «любое значение» 

[:a :b _0] ← третий элемент любой
[_0 _0 _0] ← три любых одинаковых значения
[_0 _1 _2] ← три любых значения

Logic variable

fresh предоставляет «свежие» переменные: 

(fresh [x y z] …)

run/run*

  • точка входа core.logic
  • (run N …) возвращает [0…N] значений
  • (run* …) возвращает [0…∞] значений

Goal

Почти функция: 

(f  &args) → result
(f° &args) → success/failure

Goal


(+  1 2) → 3
(+° 1 2 3) → success
(+° 1 2 4) → failure
(+° 2 2 4) → success

Goal

Преобразование функции в goal: 

(f  &args) → result
(f° &args result)
success, когда result = (f &args) и failure в обратном случае

Goal


(+ 1 2) ; 3
(run* [q] (fd/+ 1 2 q)) ; (3)

Goal

Больше, чем функция: 

(run* [q] (fd/+ 1 q 3)) ; (2)
  • goal — ограничение на аргументы
  • core.logic эффективно ищет удовлетворяющие значения

Unification

Программы на core.logic не выполняются, а унифицируются

Unification


(== q [a b])
  → q ~ ([_0 _1])

Unification


(== a b)
(== q [a b])
  → q ~ ([_0 _0])

Unification


(== a b)
(== a 1)
(== q [a b])
  → q ~ ([1 1])

Unification

Порядок не важен: 

(== q [a b])
(== a 1)
(== a b)
  → q ~ ([1 1])
Это не «присваивание», а ограничение.

Unification

Сложность и вложенность могут быть любыми: 

(run* [q]
  (fresh [x y]
    (== y :a)
    (== q [x [1 y 3] x])))
;; ([_0 [1 :a 3] _0])

conde

Когда вариантов более одного: 

(run* [q]
  (conde
     [(== q 1)]
     [(== q 2)]))
;;(1 2)

conde


(run 1 [q]
  (conde
     [(== q 1)]
     [(== q 2)]))
;;(1) — порядок не определён

conde

О бесконечности: 

(run 1 [q] (membero 1 q))
;; ((1 . _0))

conde

О бесконечности: 

(run 2 [q] (membero 1 q))
;; ((1 . _0)
;;  (_0 1 . _1))

conde

О бесконечности: 

(run 3 [q] (membero 1 q))
;; ((1 . _0)
;;  (_0 1 . _1)
;;  (_0 _1 1 . _2))

(run* [q] (membero 1 q))  ; не завершается

finite domains

Иногда нужно задать принадлежность к конечному домену: 

(run* [q]

finite domains

Арифметика: 

(run* [q]
  (fd/in q (fd/interval 0 100))
  (fd/+ q q 10)) ; q + q = 10
;; (5)

finite domains


(run* [q]
  (fd/in q (fd/interval 0 100))
  (fd/+ q q q)) ; q + q = q
;; (0)

finite domains


(run* [a b]
  (fd/in a b (fd/interval 0 100))
  (fd/+ a b 3)
  (== q [a b]))
;; ([0 3] [1 2] [2 1] [3 0])
